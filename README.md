¿Cual es la diferencia principal que notaste entre for y while?

La principal diferencia que noté entre el for y el while es que el for se usa más cuando ya sabes cuántas veces quieres repetir algo, como recorrer una lista o un rango de números. En cambio, el while se usa cuando no sabes exactamente cuántas veces se va a repetir, porque depende de una condición.

¿Te pareció útil el uso de break y continue?¿Por qué? 

Sobre el uso de break y continue, sí me parecieron útiles. El break sirve para salir del ciclo antes de que termine, por ejemplo cuando ya encontraste lo que buscabas. Y el continue ayuda a saltarse una vuelta del ciclo sin detener todo, lo que es útil cuando quieres ignorar ciertos casos, como en el ejemplo de no mostrar "Juan".
